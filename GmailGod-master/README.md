# GmailGod
Multiple Gmail Dictionary Attack
## Features
- Multi-Thread
- Anonymous attack (TOR)
- Install all dependencies
### Usage
```
$ git clone https://github.com/aryanrtm/GmailGod
$ cd GmailGod
$ chmod +x gmailgod.sh
$ service tor start
$ ./gmailgod.sh
```
