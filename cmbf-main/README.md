#### Install script on Termux
```bash
$ apt update && apt upgrade
$ apt install python2
$ pip2 install mechanize
$ pip2 install requests bs4
$ apt install git
$ rm -rf cmbf
$ git clone https://github.com/Yayan-XD/cmbf
```
#### Run script
```bash
$ cd cmbf
$ python2 cmbf.py
```
