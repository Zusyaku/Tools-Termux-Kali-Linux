# Insta Fucker
Multiple Instagram Dictionary Attack
### Features
- Multi-Thread
- Anonymous attack (TOR)
- Install all dependencies
### Usage
```
$ git clone https://github.com/aryanrtm/instafuck3r
$ cd instafuck3r
$ chmod +x instafuck3r.sh
$ service tor start
$ ./instafuck3r.sh
```
