how to install and usage?<br>
**Termux:**
* `pkg install python2`
* `pip2 install mechanize`
* `pkg install git`
* `git clone https://github.com/Ranginang67/M-dork`
* `cd M-dork`
* `python2 mdork.py`

**Linux:**
* `apt-get install git`
* `apt-get install python`
* `apt-get install python-pip`
* `pip install mechanize`
* `git clone https://github.com/Ranginang67/M-dork`
* `cd  M-dork`
* `python mdork.py`<br>
