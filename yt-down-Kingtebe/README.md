## yt-down
```
Sebuah tools untuk mendownload video youtube
lewat termux / terminal linux :v
```
## Installing
```php
$ pkg update && pkg upgrade
$ pkg install git
$ pkg install php
$ git clone https://github.com/Kingtebe/yt-down
$ cd yt-down
$ php yt.php
```
