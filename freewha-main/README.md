# Install
```php
$ pkg update && pkg upgrade
$ pkg install git
$ pkg install python3
$ python3 -m pip install requests
$ python3 -m pip install bs4
$ pip install prettytable
$ git clone https://github.com/hekelpro/freewha
$ cd freewha
$ python run.py
```
