## Info-Gempa
```
Tools Information Gempa Terkini Dari BMKG
Semoga Bermanfaat :v
```
## Usage
```php
$ pkg update && pkg upgrade
$ pkg install git
$ pkg install python
$ pip install requests
$ git clone https://github.com/Kingtebe/Info-Gempa
$ cd Info-Gempa
$ python gempa.py
```

- requests
